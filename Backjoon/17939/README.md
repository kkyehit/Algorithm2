# Gazzzua
### 17939
***
- 구현
	+ max heap( priority_queue )를 이용하여 가장 큰 값의 코인을 우선으로 선택한다.
	+ 선택된 코인의 위치까지 모든 코인을 판매한다.
		- 이전에 판매한 코인의 다음 코인부터 판매하며 만약 선택된 코인의 위치가 이전에 판매한 코인의 범위 안에 포함되면 아무것도 하지 않는다.
		- 판매한 코인이 N개가 넘어가면 탐색을 종료하고 출력한다.
	+ 수익을 출력한다.
