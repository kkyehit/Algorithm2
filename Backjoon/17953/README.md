# 디저트
### 17953
***
- dp를 이용하여 해결
	* cache[i][j] = i번째 날에 전날 먹은 디저트가 j번 쨰 디저트일 경우의 최대값
		- 전날 먹을 디저트와 오늘의 디저트가 같으면 만족도를 2로 나눈 값을 더한다.
		- 다르면 그냥 더한다.
	* 첫 날에는 전날 먹을 디저트가 없으므로 [0, N] 일 떄 각각 탐색을 수행하고 최대값을 출력한다.
	
- 제6회 한양대학교 프로그래밍 경시대회 Advanced Division E번
